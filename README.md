# Student Wallet

A new Flutter project designed to help students manage their finances efficiently.

## Description

Student Wallet is a mobile application built with Flutter that provides students with a simple and intuitive interface to manage their money. Users can track their expenses, view transaction histories, manage linked cards, and get insights into their spending habits through reports.

## Features

- **Home Screen**: A central dashboard to view recent transactions and quick actions.
- **Cards Management**: View and manage linked bank accounts or cards.
- **Transaction History**: A detailed list of all past transactions.
- **User Profile**: Manage user information and application settings.
- **Spending Reports**: Visualize spending habits with progress indicators and reports.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have the Flutter SDK installed on your machine. For installation instructions, please see the [official Flutter documentation](https://flutter.dev/docs/get-started/install).

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Rasheduzzaman-Rochi/assignment-two-flutter_UI-Rasheduzzaman-Rochi-2320366
    ```
2.  Install dependencies
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```

## Folder Structure

The project follows a feature-first structure to keep the code organized and scalable.

```
lib/
├── features/
│   ├── cards/
│   ├── home/
│   ├── navigation/
│   ├── profiles/
│   └── reports/
└── main.dart
```

-   `main.dart`: The entry point of the application.
-   `features/`: Contains all the application features, separated into modules. Each feature folder typically contains `screens` and `widgets` sub-folders.
